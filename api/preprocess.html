preprocess module¶

This is a script to pre-process the scanning data, it assumes that the input is multiple 2D slice images and creates a 3D image from the input.

The output is normalized (mean = 0 and std = 1) and only the slice, defined by the mask bounding box, is extracted and then resized to a 64x64x64 3D image.

To do so it uses the utils.ScanNormalizer class so it this file only parses the arguments and then calls the previous class.
